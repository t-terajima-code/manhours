以下の仕様書に従い、実績工数データとExcelの損益データを用いて、業務ごとの「投入人員」「労務費」「経費」を按分計算するスクリプト(`allocate_costs.py`)を作成してください。

# 1. 実行環境・制約
- **使用ライブラリ:** 標準ライブラリ（`csv`, `os`, `argparse`, `glob` 等）に加え、Excel読み込み用に `pandas`, `openpyxl` の使用を許可する。
- **文字コード:** 入出力のCSVファイルおよびスクリプトファイル自体はすべて `Shift-JIS (cp932)` を前提とする。
- **実行方法:** コマンドライン引数 `--month` で対象年月（例: `202604` または `2026/04`）を必須で指定させる。

# 2. ディレクトリ構造と環境設定ファイル (`env`)
カレントディレクトリにある設定ファイル（デフォルトは `env`）からパスを読み込む。
- 1行目: ルートディレクトリの絶対パス
- 3行目: `member` ディレクトリ（入力工数データと出力ファイルの場所）
※ `soneki` ディレクトリ（損益データの場所）はルート直下に結合して作成する。
※ パス結合時は `os.path.join` のバグを防ぐため、子ディレクトリ名の先頭の `\` や `/` を `.lstrip('\\/')` で除去すること。

# 3. 損益データ（Excel）からの金額取得
`soneki` ディレクトリ内の `.xlsx` ファイル（一時ファイル `~$` は除外）を検索し、最初に見つかったファイルを対象とする。

## 3-1. 対象シートの自動選択
引数 `--month` の末尾2桁からゼロ埋めを外し、「月」を付けた文字列（例: `202604` → `4月`）をシート名として `pandas` で読み込む。

## 3-2. 金額の動的抽出ロジック
Excel内のレイアウトから、以下の手順で金額を動的に特定する。
1. **ターゲット列の特定:** 行の値を走査し、「全社」という文字を含む列を見つけ、それより右側で最初に「実績金額」という文字が出現する列をターゲット列とする。
2. **労務費の取得:** 行を下に走査し、「(14)機能部間接部門費（労務費）」が含まれるブロックを見つける。そのブロック内にある「研究開発」の行のターゲット列の数値を「労務費」とする。
3. **経費の取得:** 同様に「(15)機能部間接部門費（経費）」のブロックを見つけ、その中の「研究開発」の行のターゲット列の数値を「経費」とする。
※取得に失敗した場合のフェイルセーフとして、労務費=17836.0、経費=3929.0 をテスト値として返すこと。

# 4. 実績工数 (`row_data.csv`) の読み込みと按分母数の算出
`member` ディレクトリの `row_data.csv` を読み込み、以下の2つの母数を算出する。

## 4-1. 有効担当者数（投入人員の母数）
指定月の実績データにおいて、区分に関わらず **「月間の合計工数が 0 より大きい（1以上の実績がある）」担当者** を抽出し、その人数を1人＝1としてカウントした合計を「有効担当者数」とする。（休職者などの工数0の人物を分母から除外するため）。

## 4-2. 対象工数（按分比率の分母）
指定月の実績データのうち、以下のいずれかの条件を満たすレコードのみを「按分対象」として抽出し、その合計工数を「按分の分母」とする。
- 内外製区分が `N`, `G`, `K` のいずれか
- 内外製区分が `y` (または `Y`) であり、かつ 品区コードが `y1`, `y10`, `y20` のいずれか

# 5. 按分計算と出力の仕様
- **出力先:** `member` ディレクトリの `[年月]_allocated_costs.csv`
- **ヘッダー構成:** `['月', '案件（業務）名', '内外製区分', '品区コード', '対象工数(hour)', '按分比率', '投入人員(人)', '労務費(千円)', '経費(千円)']`
- **按分ロジック:** - 按分比率 = `各対象レコードの工数 / 4-2で求めた対象工数の合計`
  - 投入人員 = `按分比率 * 有効担当者数`
  - 労務費 = `按分比率 * Excelから取得した労務費`
  - 経費 = `按分比率 * Excelから取得した経費`
- **数値の丸め:** 工数・労務費・経費は小数第2位、按分比率は小数第6位、**投入人員は小数第4位** とする。
- **フッター（末尾行）:**
  1. 空行を1行挟む
  2. `[合計]` 行として、各数値の合計を出力（比率は1.000000）。
  3. `[設定値]` 行として、名称（`総対象工数 / 有効担当者数 / 労務費 / 経費 ({ファイル名})`）と、各母数（対象工数合計、有効担当者数、取得した労務費、取得した経費）を並べて出力する。
