以下の仕様書に従い、実績工数データと勤怠システムから出力された勤怠データを用いて、担当者ごとの実働按分時間および「人工数」を計算するスクリプト(`allocate_actual_hours.py`)を作成してください。

# 1. 実行環境・制約
- **使用ライブラリ:** Pythonの標準ライブラリのみ（`csv`, `os`, `argparse`, `glob`, `re`, `html`, `io` 等）。`pandas` や `BeautifulSoup` 等の外部ライブラリは一切使用禁止。
- **文字コード:** 入出力ファイルおよびスクリプトファイル自身はすべて `Shift-JIS (cp932)` を基本とするが、勤怠データはBOM付きUTF-8などで出力されることがあるためエンコーディングフォールバックを実装すること。
- **実行方法:** コマンドライン引数 `--month` で対象年月（例: `202604` または `2026/04`）を必須で指定させる。

# 2. ディレクトリ構造と環境設定ファイル (`env`)
カレントディレクトリにある設定ファイル（デフォルトは `env`）からパスと設定値を読み込む。
- 1行目: ルートディレクトリの絶対パス
- 3行目: `member` ディレクトリ（入力工数データと出力ファイルの場所）
- 4行目: `kintai` ディレクトリ（勤怠データの場所）
- 7行目: 標準時間（例: `7.75` など。休暇日数を時間に換算するための数値）
※パスの結合時は `os.path.join` のバグを防ぐため、子ディレクトリ名の先頭の `\` や `/` を `.lstrip('\\/')` で除去すること。

# 3. 勤怠データ (`*{month}*.*`) の読み込み仕様
`kintai` ディレクトリ内から、指定月を含む `.xls`, `.html`, `.csv` ファイルを検索して最初に見つかったものを使用する。
- **データ形式の罠への対応（超重要）:**
  拡張子が `.xls` であっても中身がHTMLテーブルであるケースに対応するため、読み込んだテキストに `<html` や `<table` が含まれる場合は正規表現（`re`）と `html.unescape` を用いてタグを除去し、テキストリストとして解析すること。通常のCSVの場合は `csv.reader` を用いる。
- **取得対象:**
  - `名前` と `実働時間` の列を探し、名前が空欄でない行（合計行を避けるため）の `実働時間` を月間総実働時間として合算する。
  - 以下の20項目の休暇・欠勤等の列を探し、項目ごとに日数を合算する。（HTMLの場合、ヘッダー名に「日数」が付与されている場合があるため考慮すること）。
  [対象20項目]: 有休, 特別有給休暇, 積立有休, 代休, 欠勤, 子の看護休暇, 介護休暇, 産前産後休業, 育児休業, 介護休業, 公休, 産後パパ育休, 生理休暇, 組欠, 無断欠勤, 公傷, 振替休日, 傷病休暇, 私用外出, 休職

# 4. 工数データ (`row_data.csv`) の読み込みと母数算出
`member` ディレクトリの `row_data.csv` を読み込む。
（レイアウト: 1列目:月, 2列目:案件名, 3列目:区分, 4列目:品区, 5列目以降:各担当者の工数）

- **有効担当者数の算出:**
  指定月の実績データにおいて、区分に関わらず「月間の合計工数が 0 より大きい（1以上の実績がある）」担当者を抽出し、その合計人数を「有効担当者数」とする。
- **除外条件:** 引数の月と一致しない行、および「内外製区分」と「品区コード」が両方とも空欄の行は按分計算の対象外としてスキップする。
- **集計:**
  行ごとの合計工数と、月間の総合計工数（按分の分母）を計算する。

# 5. 按分計算と出力の仕様
- **出力先:** `member` ディレクトリの `[年月]_allocated_hours.csv`
- **動的ヘッダー構成:** `['月', '案件（業務）名', '内外製区分', '品区コード', '合計工数(hour)', '全体比率', '人工数', '合計按分時間(hour)']` に続き、抽出した担当者名ごとに `[担当者名]_按分時間(hour)` を並べる。
- **按分ロジック:**
  - 全体比率 = `対象業務の合計工数 / 月間総合計工数`
  - 人工数 = `全体比率 * 有効担当者数`
  - 合計按分時間 = `全体比率 * 月間総実働時間`
  - 担当者ごとの按分時間 = `(個人の対象工数 / 月間総合計工数) * 月間総実働時間`
- **数値の丸め:** 工数は小数第2位、比率は小数第6位、人工数および按分時間は小数第4位とする。
- **フッター（末尾行）:**
  1. 空行を1行挟む
  2. `[合計]` 行として、工数・比率(1.000000)・人工数・各按分時間の合計値を出力。
  3. `[設定値]` 行として、`総実働時間({読み込んだ勤怠ファイル名})` とその合計時間を出力。
  4. `[設定値]` 行として、`有効担当者数` とその人数を出力。
  5. 続いて `[設定値]` 行として、対象20項目の休暇等すべてを縦に並べて出力。
     フォーマット: `[設定値], {休暇名} 換算時間({日数}日×{標準時間}h), '', '', '', '', '', {日数 * 標準時間 (小数第4位)}`
